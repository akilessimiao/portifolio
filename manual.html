⚡ Tanque Digital
Sistema Integrado de PDV + E-commerce

📋 Visão Geral

O Tanque Digital é uma plataforma completa para gestão de vendas físicas e online, com emissão fiscal, pagamentos via PIX e relatórios automatizados.

🏪 PDV Local

Operações em loja física com múltiplos perfis (admin, operator, cashier)

🛍️ E-commerce

Loja virtual integrada com estoque e fiscal

💰 PIX

Pagamentos instantâneos via Mercado Pago

📄 NFC-e

Emissão fiscal integrada com SEFAZ/RN

📱 WhatsApp

Relatórios e notificações automáticas

🔒 Segurança

AES-256, CSRF, validação de licença

📁 Estrutura do Banco de Dados

Principais tabelas do sistema:

produtos        - Catálogo de produtos
clientes        - Cadastro de clientes
pedidos         - Vendas (online e PDV)
itens_pedido    - Itens de cada venda
nfe             - Notas fiscais emitidas
usuarios        - Controle de acesso (admin, operator, cashier)
logs            - Logs de sistema

🔧 Configuração Rápida

Pré-requisitos

  • PHP 7.4+
  • MySQL 5.7+
  • Apache/Nginx
  • Node.js (para PDV local)

Comandos básicos

# Clone o repositório
git clone https://github.com/seu-usuario/tanque-digital.git

# Configure o banco
mysql -u root -p -e "CREATE DATABASE pdvtanque;"

# Instale dependências
composer install
npm install

# Configure ambiente
cp .env.example .env
# Edite .env com suas credenciais
