🚀 Getting Started & Installation

Requirements

  • PHP 8.1+ & Composer
  • Node.js (LTS version) & NPM
  • MySQL 5.7+ / MariaDB / Supabase

Base Deployment Commands

# Clone the repository
git clone https://github.com/your-user/tanque-digital.git

# Create the local database
mysql -u root -p -e "CREATE DATABASE pdvtanque;"

# Install backend & frontend dependencies
composer install
npm install

# Configure environment variables
cp .env.example .env
# Open .env and add your valid database & third-party keys
